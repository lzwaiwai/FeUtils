# fe-utils v1.0.0

## Installation

#### In a browser:
```html
<script src="fe-utils.js"></script>
```

#### Using npm:
```shell
$ npm i -g npm
$ npm i --save fe-utils
```

#### Usage:
```js
// Load the full build.
var feUtils = require('fe-utils')

// Load method categories.
var cookie = require('fe-utils/cookie')
```
